# UI Styles Reference — 67 Styles

## General Styles (49)

### 1. Minimalism & Swiss Style
Clean lines, whitespace, systematic grids. Best for enterprise apps, dashboards, documentation.
Keywords: Grid system, negative space, functional, precise spacing, monochrome accents.

### 2. Neumorphism
Soft shadows creating extruded/inset effects. Best for health/wellness, meditation apps.
Keywords: Inner shadows, soft UI, pastel backgrounds, subtle depth, rounded corners.

### 3. Glassmorphism
Frosted glass effect with blur and transparency. Best for modern SaaS, financial dashboards.
Keywords: backdrop-filter blur, translucent panels, layered depth, vibrant backgrounds.
CSS: `background: rgba(255,255,255,0.1); backdrop-filter: blur(20px); border: 1px solid rgba(255,255,255,0.2);`

### 4. Brutalism
Raw, intentionally rough aesthetics. Best for design portfolios, artistic projects.
Keywords: Thick borders, monospace fonts, raw HTML feel, high contrast, no polish.

### 5. 3D & Hyperrealism
Depth, perspective, realistic materials. Best for gaming, product showcase, immersive.
Keywords: Three.js, WebGL, realistic lighting, depth of field, material design.

### 6. Vibrant & Block-based
Bold color blocks, chunky sections. Best for startups, creative agencies, gaming.
Keywords: Large color fills, geometric shapes, strong typography, card-heavy layouts.

### 7. Dark Mode (OLED)
True black backgrounds, neon accents. Best for night-mode apps, coding platforms.
Keywords: #000000 background, low-brightness accents, OLED-optimized, reduced eye strain.

### 8. Accessible & Ethical
Inclusive design first. Best for government, healthcare, education.
Keywords: WCAG AAA, large touch targets, clear hierarchy, semantic HTML, screen reader ready.

### 9. Claymorphism
3D clay-like rounded elements. Best for educational apps, children's apps, SaaS.
Keywords: Rounded blobs, pastel colors, subtle 3D rotation, playful shadows, soft edges.

### 10. Aurora UI
Northern lights color gradients. Best for modern SaaS, creative agencies.
Keywords: Flowing gradients, animated backgrounds, cosmic palette, ethereal glow.

### 11. Retro-Futurism
Past meets future. Best for gaming, entertainment, music platforms.
Keywords: Neon on dark, CRT effects, chrome typography, synthwave colors.

### 12. Flat Design
No shadows, no gradients, pure shapes. Best for web apps, mobile apps, MVPs.
Keywords: Solid colors, simple icons, clean typography, no depth effects.

### 13. Skeuomorphism
Realistic textures mimicking physical objects. Best for legacy apps, gaming, premium.
Keywords: Leather textures, metal gradients, realistic shadows, physical metaphors.

### 14. Liquid Glass
Premium translucent surfaces. Best for premium SaaS, high-end e-commerce.
Keywords: Refraction effects, crystal clarity, dynamic transparency, light play.

### 15. Motion-Driven
Animation IS the design. Best for portfolio sites, storytelling platforms.
Keywords: Page transitions, scroll-linked animations, kinetic typography, choreographed reveals.

### 16. Micro-interactions
Small delightful responses. Best for mobile apps, touchscreen UIs.
Keywords: Button ripples, toggle animations, pull-to-refresh, haptic feedback cues.

### 17. Inclusive Design
Universal access. Best for public services, education, healthcare.
Keywords: Multi-language, cultural sensitivity, cognitive load reduction, flexible input.

### 18. Zero Interface
No visible UI. Best for voice assistants, AI platforms.
Keywords: Conversational, ambient, context-aware, gesture-based, invisible computing.

### 19. Soft UI Evolution
Modern take on neumorphism. Best for modern enterprise, SaaS.
Keywords: Subtle elevation, soft depth, muted palette, premium feel, organic shapes.

### 20. Neubrutalism
Brutalism meets Gen Z. Best for Gen Z brands, startups, Figma-style.
Keywords: Black borders, bright fills, offset shadows, chunky buttons, playful.

### 21. Bento Box Grid
Apple-inspired grid cards. Best for dashboards, product pages, portfolios.
Keywords: Multi-size cards, rounded corners, information density, visual hierarchy.

### 22. Y2K Aesthetic
2000s nostalgia. Best for fashion, music, Gen Z.
Keywords: Chrome, bubble fonts, gradient meshes, star sparkles, metallic.

### 23. Cyberpunk UI
Neon dystopia. Best for gaming, tech products, crypto.
Keywords: Neon green/pink, glitch effects, HUD elements, terminal fonts, scan lines.

### 24. Organic Biophilic
Nature-inspired. Best for wellness, sustainability brands.
Keywords: Earthy tones, organic shapes, leaf motifs, natural textures, green palette.

### 25. AI-Native UI
Designed for AI products. Best for chatbots, copilots, AI tools.
Keywords: Streaming text, typing indicators, suggestion chips, context panels, gradient orbs.
