# 2026-02-04 — Portal Tema/Live Yazı Rengi + Logo Konum Kontrolleri

## Özet
Bugün Portal tarafında tema uyumluluğu ve edit ekranı kontrolleri iyileştirildi:

- **Live (sunum) ekranında** Join / Tek tek (Rotate) / Öne çıkan (Spotlight) görünümlerinde hard-coded yazı renkleri kaldırılıp tema `textColor` (ve türevleri) ile uyumlu hale getirildi.
- **Edit ekranında** logo ayar panelinin vurgu rengi tema `primaryColor`’a bağlandı (Slider + kaldır butonu vb.).
- Ayrıca Theme panelinde görünen basit **🖼️ Logo** bölümünde eksik olan **X Konum / Y Konum** sliderları eklendi (senin ekranda görünmeyen kısım buydu).

## İlgili dosyalar
- Live ekran (metin rengi + embed/preview iyileştirmeleri):
  - `site/apps/portal/src/app/(presentation)/events/[id]/live/page.tsx`
- Edit ekran (tema paneli + logo ayarları + X/Y):
  - `site/apps/portal/src/app/(dashboard)/events/[id]/edit/page.tsx`

## Prod deploy (uygulandı)
Portal değişikliklerini canlıya almak için:

- `cd /srv/webhosting/soruyorum.online/site`
- `docker compose -f docker-compose.yml -f docker-compose.traefik.yml up -d --build soruyorum-portal`

Not: Compose bake yüzünden `soruyorum-api-server` image’ı da build edilebilir.

## Doğrulama
- `/events/<id>/edit` içinde Theme/Logo bölümünde **Logo Boyutu** altında **X Konum** ve **Y Konum** sliderları görünmeli.
- Live ekranında (Join / Tek tek) yazı rengi tema `textColor` değişince anında uyumlu olmalı.

## Yarın kaldığımız yer
- İstersen edit ekranındaki Join/Duvar/Tek tek preview switcher butonlarının aktif/hover renklerini de tema `primaryColor` ile tamamen uyumlu hale getirebiliriz.
